# Introduction to Python

Welcome to your first Python lesson! In this lesson, you'll learn the fundamentals of Python programming.

## What is Python?

Python is a **high-level**, **interpreted** programming language known for:

- 🎯 Simple and readable syntax
- 🚀 Powerful standard library
- 🌍 Large community support
- 📦 Extensive third-party packages

## Your First Python Program

Let's start with the classic "Hello World" program:

```python
print("Hello, World!")
```

### How it works:
1. `print()` is a built-in function
2. It displays text to the console
3. Strings can use single `'` or double `"` quotes

## Variables and Data Types

Python supports various data types:

```python
# Numbers
age = 25
price = 19.99

# Strings
name = "Alice"
message = 'Hello Python!'

# Booleans
is_student = True
has_license = False

# Lists
fruits = ["apple", "banana", "cherry"]

# Dictionaries
person = {
    "name": "John",
    "age": 30,
    "city": "New York"
}
```

### Key Points:
- Variables don't need type declarations
- Python automatically infers the type
- Variable names should be descriptive

## Basic Operations

### Arithmetic Operators

```python
a = 10
b = 3

addition = a + b        # 13
subtraction = a - b     # 7
multiplication = a * b  # 30
division = a / b        # 3.333...
floor_division = a // b # 3
modulus = a % b         # 1
exponent = a ** b       # 1000
```

### String Operations

```python
first_name = "John"
last_name = "Doe"

# Concatenation
full_name = first_name + " " + last_name

# Repetition
laugh = "Ha" * 3  # "HaHaHa"

# String formatting
greeting = f"Hello, {first_name}!"
```
